# 🎬 光影导航 · 免费正版影视资源站

一个纯静态的影视导航网站：**搜片名，一步直达各免费正版平台的该片搜索结果**，不用先跳平台主页再手动查找。

## ✨ 功能
- 🔍 **搜索直达**：输入片名（如「逃学威龙」），列出各免费平台的「去XX看《片名》」按钮，直达该平台这部片的搜索结果页。
- 🏷️ **分类筛选**：电影 / 电视剧 / 纪录片 / 动漫 / 公版经典。
- 📺 **分辨率筛选**：1080P / 2K / 4K（选 2K 会同时列出支持 2K 与 4K 的平台）。
- 💡 **示例片名**：点击即搜，含逃学威龙、流浪地球、BBC纪录片等。

## ⚖️ 合规说明
本站**仅聚合官方免费 / 免费专区 / 公版资源**，**不存储、不内嵌任何片源**，所有观看均在对应平台官方页面完成。内容版权归原平台与权利人所有，支持正版。

## 🛠️ 技术
纯静态站点，零依赖、零构建：
- `index.html` — 页面结构
- `styles.css` — 样式
- `app.js` — 筛选 / 搜索 / 直达逻辑
- `data.js` — 平台数据（含各平台「按片名搜索」的直达链接模板）

## 🚀 本地预览
直接用浏览器打开 `index.html` 即可；或起一个本地服务：
```bash
python -m http.server 8080
# 浏览器访问 http://localhost:8080
```

## 📦 部署（GitHub Pages）
1. 推送到 GitHub 仓库。
2. 仓库 Settings → Pages → Source 选 `main` 分支根目录 → Save。
3. 稍候即可通过 `https://<用户名>.github.io/<仓库名>/` 访问。

## ➕ 如何新增平台
编辑 `data.js` 的 `PLATFORMS` 数组，照现有格式加一条即可：
```js
{
  id: 'new',
  name: '平台名',
  emoji: '🎨',
  desc: '一句话简介',
  cats: ['movie', 'tv'],     // movie/tv/doc/anime/public
  res: '1080p',              // 1080p/2k/4k
  note: '补充说明',
  searchTpl: 'https://平台搜索地址?q={q}'  // {q} 会被替换为编码后的片名
}
```
